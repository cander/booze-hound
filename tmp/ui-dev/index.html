What is Booze Hound?

Are you an Oregonian who has used the Oregon Liquor & Cannabis Commission's (OLCC) Oregon Liquor Search site but you've been frustrated with the experience? Welcome to Booze Hound!

Booze Hound is an improved user experience for Oregonians who use the Oregon Liquor & Cannabis Commission's (OLCC) Oregon Liquor Search site.

The additional functionality and improved user interface of Booze Hound allows users to:

  • Create an account for a customized experience
  • Follow one or more stores for updates to stock
  • Follow one or more categories of liquor for updates to stock
  • ...and more!
